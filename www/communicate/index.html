How to Communicate?

First, request to join the Jersey project in the role of an Observer.

Subscribe to the news feed to keep informed of project announcements.

If you are or plan to be a user of Jersey subscribe to the user email list.

If you plan to contribute to Jersey or are interested in the discussions on the developement of Jersey subscribe to the dev email list.

Subscribe to the commits list to get notified of changes to the source code.

Issues can be sumitted and tracked by using the Issue Tracker and/or by subscribing to the issues email list.

The source for Jersey is accessible from the Subversion respository at the following location: https://jersey.dev.java.net/svn/jersey/trunk/jersey
