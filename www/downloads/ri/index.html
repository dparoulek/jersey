The JSR 311 Reference Implementation

This page provides links to download the binary and source distributions of the Reference Implementation (RI) for JSR 311: JAX-RS: The Java API for RESTful Web Services, which represents a subset of the Jersey project.

To utilize Jersey distributions, the recommended approach to getting started and using JAX-RS, please goto the Jersey project page and click on the "Start", "Download" or "Download Now" buttons.
